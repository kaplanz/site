I was today[^26] years old when I learned that I can use `<C-t>` on Unix systems
to send `SIGINFO` to a running process to request it to print info to `stdout`.

This worked for `dd` and `sleep`[^cmd], which both print their progress (blocks
copied and time left respectively). Neat!

[^cmd]: Being the only commands I've tested before I went to write this.
